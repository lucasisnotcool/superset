# is required for full Wren parity and is installed by default.

# LanceDB-backed persistent embedding index (WREN_VECTOR_INDEX=lancedb). Survives

# need the separate `pylance` package, so the code uses `to_arrow()`. Embedding
# retrieval degrades to the in-process index when this is absent.

# Document extractors for the uploaded-document RAG suite

# directly -- no Microsoft Office / LibreOffice / COM, so they run on the
# Office-less Linux deploy. The code import-guards their absence and records the
# document with status="error" and a clear message, so they are only required to
